# Excel完全対応版

GitHubへ app.py と requirements.txt を上書きしてください。

## Excelを使う
1. アプリ下部の「現在のデータをExcelで保存」を押す
2. ExcelでScheduleやBudgetSettingsを編集する
3. アプリの「編集したExcelを読み込む」から反映する
4. 画面で予算を変更した後も、Excelを再ダウンロードできる

## シート
- Schedule: 予定、日時、順番、予算、メモ
- RegisteredPlaces: 登録スポット
- BudgetSettings: ガソリン、高速、駐車場、お土産など
- TripSettings: 旅行日、開始時間、移動手段
- Dashboard: 予算集計

JSONは移行用として残しています。
